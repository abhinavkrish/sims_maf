This directory contains a variety of example MAF config scripts.

To run, after configuring your environement:
>runDriver.py <driverCfg.py>

The Config files:
----------------
allBinnerCfg.py:  This file contains at least one example of every type of binner available in MAF

starDriver.py:  An example that re-creates most of the plots in an SSTAR report, but uses the healpixBinner.

starDriver_fields.py: Re-creates the plots in SSTAR report.  

starDriver_fields_fulldb.py:  Example of how one can get fieldID information from a different table in the database.

maf_example_web.py:  Driver script for making example plots for Confluence documentation.
